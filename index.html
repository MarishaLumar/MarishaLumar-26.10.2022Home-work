Селектори CSS

Універсальні селектори CSS

Універсальні селектори CSS позначаються звичайною зірочкою "*".

Під зірочкою може матися на увазі будь-який елемент розмітки HTML-сторінки.

Cелектори за назвою тега

CSS селектори за назвою тега-запис селекторів за тегами. Наприклад:
h1 { color: red; }
де h1 – селектор за назвою тега.

CSS селектори за класами

CSS селектори за класами-«class» - це атрибут HTML-тегу, він дуже зручний тим, що може застосовуватися до однотипних елементів.
Назва класу може бути будь-яким, головне, щоб воно відповідало тому елементу
Aтрибут "class" вважається найголовнішим інструментом CSS. Під час створення CSS-правила точка «.» перед ім'ям класу є обов'язковим.

CSS селектори по id

  • Атрибут id застосовується до індивідуального елементу, елемент з id може бути в єдиному екземплярі на сторінці згідно правилам семантики. У каскадних таблицях стилів ці атрибути використовуються вкрай рідко, їх застосування до них взагалі не вітається, вони призначені в основному для роботи з javascript. CSS селектори за атрибутами
  • Селектори атрибутів відбирають елементи наявності атрибута або його значення.

CSS селектори нащадків, або контекстні селектори

Це коли один елемент HTML-сторінки вкладений в інший, причому не обов'язково він має бути першим нащадком.

Дочірні селектори CSS

  • Дочірні селектори CSS відрізняються від селекторів нащадків тим, що під дочірнім мається на увазі лише перший нащадок, ніякі «онуки» та «правнуки» не потрапляють до цієї категорії. За допомогою сестринського селектора створюється CSS-правило для HTML-елементів, розташованих на одному рівні, причому той елемент, який записується як сестринський повинен бути найближчим із себе подібних до того елемента, якого він записується в «сестри».

CSS селектори псевдокласів

  • До селекторів псевдокласів відносяться селектори CSS, які описують стану елементів під час здійснення над ними будь-якої дії. Це може бути наведення елемент курсору миші, клік по елементу тощо. Записуються вони через двокрапку до елемента, до якого застосовуються.

CSS селектори псевдоелементів

  1. Псевдоелемент у CSS - це ключове слово, що додається до селектора, що дозволяє стилізувати певну частину обраного елемента. Наприклад, псевдоелемент ::first-line може бути використаний для зміни шрифту першого рядку абзацу.
